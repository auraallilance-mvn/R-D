Project Overview

The goal of this project was to create an autonomous mobile robot that could navigate a room, detect obstacles in real-time, and make intelligent decisions to avoid collisions. Using the HC-SR04 ultrasonic sensor and L298N motor driver, the robot continuously scans its environment and adjusts its path accordingly.

When an obstacle is detected within 20cm, the robot stops, reverses slightly, and turns right to find a clear path. This simple yet effective algorithm demonstrates the fundamentals of autonomous navigation and forms the basis for more complex robotics applications.

Full Source Code

Complete Arduino implementation with obstacle avoidance logic:

// Motor pins (L298N)
const int motor1Pin1 = 5;
const int motor1Pin2 = 6;
const int motor2Pin1 = 9;
const int motor2Pin2 = 10;

// Ultrasonic sensor pins
const int trigPin = 3;
const int echoPin = 4;

void setup() {
  Serial.begin(9600);
  
  // Motor pins as outputs
  pinMode(motor1Pin1, OUTPUT);
  pinMode(motor1Pin2, OUTPUT);
  pinMode(motor2Pin1, OUTPUT);
  pinMode(motor2Pin2, OUTPUT);
  
  // Ultrasonic sensor pins
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  
  Serial.println("✅ Autonomous Robot Initialized");
}

void loop() {
  long duration;
  int distance;
  
  // Trigger ultrasonic pulse
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  // Read echo
  duration = pulseIn(echoPin, HIGH);
  
  // Calculate distance (cm)
  distance = duration * 0.034 / 2;
  
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  // Obstacle avoidance logic
  if (distance <= 20 && distance > 0) {
    Serial.println("⚠️ Obstacle detected! Avoiding...");
    stopMotors();
    delay(500);
    
    // Reverse
    moveBackward();
    delay(300);
    
    // Turn right
    turnRight();
    delay(500);
  } else {
    // Clear path - move forward
    moveForward();
  }
  
  delay(100);
}

void moveForward() {
  digitalWrite(motor1Pin1, HIGH);
  digitalWrite(motor1Pin2, LOW);
  digitalWrite(motor2Pin1, HIGH);
  digitalWrite(motor2Pin2, LOW);
}

void moveBackward() {
  digitalWrite(motor1Pin1, LOW);
  digitalWrite(motor1Pin2, HIGH);
  digitalWrite(motor2Pin1, LOW);
  digitalWrite(motor2Pin2, HIGH);
}

void turnRight() {
  digitalWrite(motor1Pin1, HIGH);
  digitalWrite(motor1Pin2, LOW);
  digitalWrite(motor2Pin1, LOW);
  digitalWrite(motor2Pin2, HIGH);
}

void turnLeft() {
  digitalWrite(motor1Pin1, LOW);
  digitalWrite(motor1Pin2, HIGH);
  digitalWrite(motor2Pin1, HIGH);
  digitalWrite(motor2Pin2, LOW);
}

void stopMotors() {
  digitalWrite(motor1Pin1, LOW);
  digitalWrite(motor1Pin2, LOW);
  digitalWrite(motor2Pin1, LOW);
  digitalWrite(motor2Pin2, LOW);
}

Hardware Setup

Components Required

  • Arduino Uno
  • HC-SR04 Ultrasonic Sensor
  • L298N Motor Driver Module
  • 2× DC Motors with wheels
  • Robot chassis
  • 9V Battery or power bank
  • Jumper wires

Wiring Connections

HC-SR04 Ultrasonic:

VCC → Arduino 5V

Trig → Arduino Pin 3

Echo → Arduino Pin 4

GND → Arduino GND

L298N Motor Driver:

IN1 → Arduino Pin 5

IN2 → Arduino Pin 6

IN3 → Arduino Pin 9

IN4 → Arduino Pin 10

OUT1, OUT2 → Motor 1

OUT3, OUT4 → Motor 2

12V → Battery +

GND → Battery - and Arduino GND

How It Works

1

Distance Measurement

HC-SR04 sends ultrasonic pulses and measures echo time to calculate distance

2

Decision Making

If distance ≤ 20cm, obstacle detected; otherwise, path is clear

3

Avoidance Maneuver

Stop → Reverse → Turn right to find alternative path

4

Continue Navigation

Resume forward movement and repeat scanning cycle
