Project Overview

This NeoPixel scrolling display project creates a customizable LED matrix that can be controlled from anywhere via WiFi. The system features a web interface for real-time text updates, timing controls, and display management. Perfect for announcements, advertisements, or decorative displays.

Project Guidance & Supervision

This LED matrix project was developed under the expert guidance of Prof. M.V. Narayana Sir, Head of Department (ECE) and R&D Dean at KSRM College of Engineering. His mentorship in advanced LED control systems and web-based interfaces provided the foundation for understanding addressable LED programming and wireless display technologies.

The 34×5 LED matrix (170 NeoPixels) displays scrolling text with customizable colors and timing. The ESP8266 creates a web server that provides an intuitive interface for controlling the display remotely from any device connected to the same network.

Hardware Setup

Components Required

  • ESP8266 Development Board (NodeMCU/Wemos D1)
  • 170× WS2812B NeoPixel LEDs (34×5 matrix)
  • 5V Power Supply (sufficient for LED current)
  • Breadboard or custom PCB
  • Jumper wires and connectors
  • Frame or mounting structure

Wiring Connections

ESP8266 D6 (GPIO 12) → NeoPixel Data In

5V Power Supply → NeoPixel VCC

ESP8266 GND → NeoPixel GND

ESP8266 VIN → 5V (if using external power)

Note: Use adequate power supply for 170 LEDs

// Custom font array for LED matrix
byte font[][5] = {
  {B01110,B10001,B11111,B10001,B10001}, // A
  {B11110,B10001,B11110,B10001,B11110}, // B
  // ... more letters
};

// Main scrolling function
void scrollDisplay() {
  if (!displayOn || !inTimeWindow()) {
    clearMatrix();
    strip.show();
    return;
  }
  
  // Update scroll position
  scrollOffset--;
  if (scrollOffset < -textLenPx - WIDTH) 
    scrollOffset = WIDTH;
  
  // Render text with colors
  clearMatrix();
  int x = scrollOffset;
  for (int i = 0; i < displayText.length(); i++) {
    uint32_t col = (i < 7) ? 
      strip.Color(255, 0, 0) : strip.Color(0, 255, 0);
    drawLetter(displayText[i], x, col);
    x += 6;
  }
  strip.show();
}
