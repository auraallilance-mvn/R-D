Full Source Code

Complete Python implementation with GPIO control:

import RPi.GPIO as GPIO
import time

# Pin configuration
LED_PIN = 18  # GPIO 18 (Physical pin 12)

# Setup
GPIO.setmode(GPIO.BCM)  # Use BCM pin numbering
GPIO.setup(LED_PIN, GPIO.OUT)

print("‚úÖ Raspberry Pi GPIO LED Control")
print("Press Ctrl+C to exit")

try:
    while True:
        # Turn LED ON
        GPIO.output(LED_PIN, GPIO.HIGH)
        print("üí° LED ON")
        time.sleep(1)
        
        # Turn LED OFF
        GPIO.output(LED_PIN, GPIO.LOW)
        print("‚ö´ LED OFF")
        time.sleep(1)

except KeyboardInterrupt:
    print("\nüõë Program stopped by user")

finally:
    # Cleanup GPIO pins
    GPIO.cleanup()
    print("‚úÖ GPIO pins cleaned up")

Advanced: PWM Breathing Effect

Smooth LED fading using Pulse Width Modulation:

import RPi.GPIO as GPIO
import time

LED_PIN = 18

GPIO.setmode(GPIO.BCM)
GPIO.setup(LED_PIN, GPIO.OUT)

# Create PWM instance (100 Hz frequency)
pwm = GPIO.PWM(LED_PIN, 100)
pwm.start(0)  # Start with 0% duty cycle

print("‚úÖ PWM Breathing Effect Started")

try:
    while True:
        # Fade in (0% to 100%)
        for duty_cycle in range(0, 101, 5):
            pwm.ChangeDutyCycle(duty_cycle)
            time.sleep(0.05)
        
        # Fade out (100% to 0%)
        for duty_cycle in range(100, -1, -5):
            pwm.ChangeDutyCycle(duty_cycle)
            time.sleep(0.05)

except KeyboardInterrupt:
    print("\nüõë Stopped")

finally:
    pwm.stop()
    GPIO.cleanup()
    print("‚úÖ Cleanup complete")

GPIO Pin Modes

BCM Mode (Broadcom SOC channel):

Uses GPIO numbers (e.g., GPIO 18)

BOARD Mode (Physical pin numbers):

Uses physical pin position (e.g., Pin 12)

Key Concepts

GPIO.setmode()

Defines pin numbering scheme - BCM or BOARD mode

GPIO.setup()

Configures pin as INPUT or OUTPUT

GPIO.output()

Sets pin to HIGH (3.3V) or LOW (0V)

GPIO.cleanup()

Resets all GPIO pins to safe state - always call in finally block
